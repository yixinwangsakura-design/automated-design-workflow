# Phase 1: Problem Framing Agent

## 1. Research Planning

### Input
- Research goal
- Research hypothesis
- Research method (survey, interview, etc.)

### Output
- Structured research plan
- Interview scripts OR survey questionnaires

### Requirements
- Questions should be unbiased and open-ended
- Align with hypothesis validation
- Organized by themes

---

## 2. Research Synthesis

### Input
- Research data (transcripts, survey results)
- Participant information (matching data with specific people)

### Output
- UX insights using CIA framework 

### CIA Framework
- Claim (What)
- Insight (Why - evidence, quotes)
- Action (Now what - recommendation)

### Requirements
- Focus on user behavior (not just opinions)
- Include quotes for empathy
- Limit to 5–6 key insights
- Validate or invalidate initial hypothesis

---

## 3. Scope Definition & Prioritization

### Input
- Time constraints

### Output
- Prioritized Insights from research / feature list
- Ranking based on:
  - Impact vs Effort
  - Business value
  - User value

### Requirements
- Provide rationale like a senior PM
- Consider market trends (if available)
