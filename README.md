# Automated Product Design Workflow Agent System

## Overview
This document defines a modular AI agent system that supports a product designer’s workflow across three phases:

1. Problem Framing  
2. Design Exploration  
3. Design Prototyping  

Each phase is independent and can be used separately or sequentially.

---

## Structure

- `/agents` → Phase-based agents
- `/shared` → Global system guidelines

---

## How to Use

- Use each phase independently  
- Or combine them for end-to-end workflow  
- Copy relevant sections into AI tools (ChatGPT, Claude, etc.)
